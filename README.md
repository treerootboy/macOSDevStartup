# macOSDevStartup
macOS 开发环境初始化脚本。利用brew, npm, composer等初始化开发环境

## 安装
```bash
# 启用 touchid sudo 验证
sudo gsed -i '1iauth sufficient pam_tid.so' /etc/pam.d/sudo

curl -fsSL https://cdn.jsdelivr.net/gh/treerootboy/macOSDevStartup@master/install.sh | bash
```
